# IPAt

IPAt is a GUI tool for batch-generating IPA transcriptions of wordlists using custom rulesets.

## Installation

This project uses **FreeSimpleGUI** and **Pyperclip**

A virtual environment is recommended

### Install all dependencies:

    pip install -r requirements.txt

## Usage

To get started with IPAt, you need to select a language. If you have a conlang, head to *'Import/Create a language config'* and click *'Make new'*:
